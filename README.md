> **"想"与"做"，只差一个番茄钟的距离。**

MemoPomo 是一款专为 Windows 打造的轻量级、纯本地工作流桌面工具。它将**四象限备忘录**、**便利贴**、**可视化日历**与**番茄钟专注**融为一体，让你从记录想法到完成任务一气呵成，全程无需联网、无需账号。

## 📖 完整使用教程

### 一、主界面布局

启动后主窗口分为左右两栏：

- **左侧**：番茄钟计时区 + 「开始背单词」按钮
- **右侧上方**：可视化日历（周 / 日 / 月视图）
- **右侧下方**：四象限备忘录 + 「便利贴」按钮

---

### 二、四象限备忘录

按照「重要 × 紧急」将任务分入四个象限：

| 象限 | 含义 | 建议 |
|------|------|------|
| Q1 | 重要且紧急 | 立即处理 |
| Q2 | 重要不紧急 | 计划安排 |
| Q3 | 紧急不重要 | 委托他人 |
| Q4 | 不重要不紧急 | 择机处理或删除 |

**基本操作：**

1. 在任意象限底部输入框输入任务名，按 Enter 或点击「+」新建任务
2. **双击**卡片可编辑标题和备注
3. 点击卡片左侧 **✓** 标记为完成（自动沉底并显示删除线）
4. **拖拽**卡片到其他象限可重新分类
5. 点击卡片右侧 **🗑** 将任务移入回收站（可在设置中恢复）

**设置重复任务：**

右键点击任意卡片 → 「设置重复」，可配置：
- 重复频率：每天 / 每周（指定星期几）/ 每月（指定日期）
- 执行时间：开始时刻 + 持续时长
- 结束日期

设置后，重复任务会在日历对应日期自动生成时间块，到点触发全屏提醒。

---

### 三、便利贴

点击备忘录区域右上角的「便利贴」按钮，打开独立便利贴窗口。

**新建便利贴：**

点击窗口顶部「＋ 新建便利贴」，即可在列表顶部创建一张新便签。

**编辑：**

- 直接点击标题或正文区域即可内联编辑，内容自动保存（300ms 防抖）

**管理：**

| 按钮 | 功能 |
|------|------|
| 📌 置顶 / 已置顶 | 切换置顶状态，置顶的便签始终显示在列表最上方，背景变为淡黄色 |
| ▲ 收起 / ▼ 展开 | 折叠后只显示标题栏，节省空间 |
| 🗑 删除 | 确认后永久删除该便签 |

**调整高度：**

拖动每张便利贴底部的灰色横条，可自由调整该便签的显示高度，高度自动保存。

---

### 四、可视化日历（Timeboxing）

**视图切换：** 顶部工具栏可在「周视图」「日视图」「月视图」之间切换，点击「今天」回到当前日期。

#### 新建时间块

- **点击**日历空白区域 → 弹出编辑对话框
  - 填写标题、选择日期、开始时间、持续时长（时间步进自动对齐 15 分钟）
  - 可选择关联一个备忘任务
- **拖拽备忘卡片**到日历对应位置 → 自动创建 1 小时计划块

#### 编辑 / 移动时间块

- **单击**时间块 → 打开编辑对话框，可修改所有字段
- **拖拽**时间块到新位置 → 自动更新开始时间
- **拖拽时间块底部边缘** → 调整持续时长

#### 全屏提醒

到达时间块的开始时间时，软件自动弹出全屏提醒：

- 点击「**立即启动**」→ 按关联的番茄钟模板立即开始计时
- 点击「**推迟 N 分钟**」→ N 分钟后再次提醒（N 可在设置中修改）
- 点击「**忽略**」→ 关闭本次提醒

#### 日历布局自定义

在「设置 → 日历布局」中可调整：

- 每小时行高（30 ~ 200 px）
- 每天列宽（60 ~ 400 px）
- 显示起始小时 / 结束小时

---

### 五、番茄钟

#### 快速开始

1. 在左侧番茄钟区域，顶部下拉框选择**模板**（默认「25+5」）
2. 可选：在「关联任务」下拉框中绑定一个备忘任务
3. 点击「**开始**」启动计时

#### 悬浮窗

计时开始后，可点击「收起」将主窗口折叠为**极简悬浮窗**（始终置顶），悬浮窗支持：

- **暂停 / 继续**
- **跳过**当前段（直接进入下一工作段或休息段）
- **重置**本次会话
- **返回**主界面

#### 段落进度条

主界面和悬浮窗均显示彩色进度条：深蓝色为工作段，浅绿色为休息段，▼ 箭头指示当前所在段。

#### 自定义模板

在「设置 → 番茄钟模板」中可新建 / 编辑 / 删除模板，自由配置每个工作段和休息段的时长与顺序。

---

### 六、背单词

#### 触发方式

1. **自动触发**：每个番茄钟段落结束后自动弹出全屏背单词界面
2. **手动触发**：点击主界面左侧「开始背单词」按钮随时使用

#### 学习流程

每个单词分三步：

1. **看释义** → 根据中文释义猜测单词
2. **填写拼写** → 在输入框中拼写单词（回车确认）
3. **查看结果** → 显示正确答案和例句，点击「完成 ✓」进入下一步

#### 功能说明

| 功能 | 说明 |
|------|------|
| 左侧单词列表 | 显示全部单词及已学习次数（×N），点击可直接跳转到该单词 |
| 顺序 ↕ / 随机 ⚄ | 顶部切换按钮，控制下一个单词的选取方式 |
| 结束按钮 | 初始灰色不可用；完成当前单词后亮起，可点击退出 |

---

### 七、设置

点击主界面右上角「设置」按钮打开设置窗口，包含以下标签页：

#### 通用
- **开机自启**：开关系统自启动
- **默认推迟时长**：全屏提醒中推迟按钮的默认分钟数

#### 番茄钟模板
- 新建 / 编辑 / 删除模板
- 设置默认模板

#### 提醒方式
- **休息屏保**：段落结束后全屏显示休息提示
- **打开网址**：段落结束后自动打开指定 URL
- **打开文件**：段落结束后自动打开指定文件

#### 日历布局
- 每小时行高、每天列宽
- 日历显示的起止小时

#### 四象限颜色
- 自定义四个象限标题栏的背景颜色

#### 回收站
- 查看已删除的备忘任务
- **恢复**：将任务重新放回对应象限
- **永久删除**：从回收站中彻底删除

#### 数据备份
- **导出所有数据**：将 `data/` 目录下所有 JSON 文件打包为 zip
- **导入所有数据**：从 zip 恢复数据（会覆盖当前数据，请谨慎操作）

## 🛠️ 从源码运行

### 环境要求

- Python 3.10+
- Windows 10 / 11

### 安装步骤

```bash
git clone https://github.com/wangyuanxin2002/MemoPomo.git
cd MemoPomo
pip install PyQt6
python main.py
```
