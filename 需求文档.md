# 番茄钟 · 日历 · 备忘录 需求文档

> 参考 UI 风格：深色系（参见 ref/1.jpg），周历视图布局

---

## 一、窗口行为

| # | 需求 | 说明 |
|---|------|------|
| 1.1 | 开机自动启动 | 登录 Windows 后自动运行，无需手动打开 |
| 1.2 | 默认悬浮顶层 | 始终显示在其他窗口之上（Always on top） |
| 1.3 | 主界面状态 | 完整功能界面，包含日历、番茄钟、备忘录 |
| 1.4 | 点击"开始计时"后缩小 | 变成一个小悬浮窗，只显示倒计时数字，可自由拖动后固定位置 |
| 1.5 | 计时结束后全屏显示 | 立即覆盖整个屏幕（含任务栏），强制提醒 |
| 1.6 | 全屏提醒可手动关闭 | 点击或按键后回到主界面 / 小悬浮窗，继续下一段 |

---

## 二、番茄钟

| # | 需求 | 说明 |
|---|------|------|
| 2.1 | 自定义时间段模板 | 例如 25+5+25+5+25+15（工作+休息交替），每段时长可自由修改 |
| 2.2 | 自选开始时间 | 不固定，每次手动选择时间或点击"立即开始" |
| 2.3 | 预计时间段显示在日历 | 计划开始的番茄钟组，在日历对应时间轴上预先显示（未来时间块） |
| 2.4 | 临近开始时弹窗提醒 | 到达预定开始时间前弹窗询问："是否现在开始？"，提供【开始计时】和【推迟 N 分钟】两个选项；N 为可编辑输入框，默认显示上次使用的历史值，可直接确认或修改后确认 |
| 2.5 | 开始时选择关联任务 | 弹出窗口供选择当前计时关联的备忘录任务；可跳过（适用于未列入备忘录的工作、运动、休息等） |
| 2.6 | 可视化倒计时 | 小悬浮窗显示：当前剩余时间、当前处于第几段（工作/休息）、任务名称（如有）；**左键单击**暂停/继续计时；**右键单击**弹出下拉菜单（见 2.10） |
| 2.10 | 悬浮窗右键菜单 | 包含以下选项：展开主界面、放弃本段/重置、跳过本段、切换关联任务、查看今日记录、设置、退出程序 |
| 2.7 | 段落切换提示 | 每段结束时提示（声音 + 全屏提醒），可确认后进入下一段 |
| 2.8 | 暂停 / 重置 | 支持暂停当前段，支持重置整个番茄组 |
| 2.9 | 完成后自动记录 | 完整完成一个番茄钟组后，实际起止时间自动写入当天日历 |

---

## 三、全屏提醒（段落结束时）

每段番茄钟结束时触发全屏提醒，支持以下模式（可在设置中配置）：

| # | 需求 | 说明 |
|---|------|------|
| 3.1 | 模式①：休息提示页 | 全屏显示"休息时间到！"，含休息倒计时，点击或计时结束后关闭 |
| 3.2 | 模式②：打开网页 | 全屏提醒同时自动打开指定网址（如冥想网站、YouTube 等） |
| 3.3 | 模式③：打开文件 | 全屏提醒同时自动打开指定本地文件（如音乐、视频、文档等） |
| 3.4 | 多模式可叠加 | 以上模式可组合，例如同时显示提示页并打开文件 |
| 3.5 | 设置中管理 | 在设置界面中配置各模式的具体网址或文件路径 |

## 四、日历（周视图）

| # | 需求 | 说明 |
|---|------|------|
| 4.1 | 周历视图 | 横轴为周一至周日，纵轴为时间轴（参考 ref/1.jpg） |
| 4.2 | 切换上周/下周/本周 | 顶部导航按钮 |
| 4.3 | 已完成时间块 | 番茄钟完成后，实际时间段以实色块显示 |
| 4.4 | 计划中时间块 | 未来已排定但尚未开始的番茄钟，以淡色/虚线色块显示 |
| 4.5 | 时间块显示任务名 | 色块上显示关联的任务名（备忘录任务或自定义名称） |
| 4.6 | 手动添加时间块 | 在日历上点击拖拽创建一段时间记录（无需通过番茄钟） |
| 4.7 | 备忘录任务拖放到日历 | 从备忘录四宫格拖放任务到日历某时间段，自动创建计划时间块 |
| 4.8 | 时间块可编辑/删除 | 点击色块可修改名称、时间范围、关联任务，或删除 |

---

## 五、备忘录（四象限）

| # | 需求 | 说明 |
|---|------|------|
| 5.1 | 四宫格布局 | 四个象限：① 重要且紧急、② 重要不紧急、③ 紧急不重要、④ 不重要不紧急 |
| 5.2 | 每个象限可添加任务 | 输入任务名称，支持简短备注 |
| 5.3 | 任务可拖动换象限 | 在四宫格内拖放，重新分类 |
| 5.4 | 任务可拖放到日历 | 从四宫格拖放到日历时间段，创建该任务的计划时间块 |
| 5.5 | 开始计时时可关联任务 | 番茄钟开始弹窗中可从四象限任务列表选择当前任务 |
| 5.6 | 任务状态标记 | 支持标记"已完成"，完成后视觉区分（如置灰或划线） |
| 5.7 | 数据持久化 | 任务保存到本地，关闭重开后不丢失 |

---

## 六、数据存储

| # | 需求 | 说明 |
|---|------|------|
| 6.1 | 本地存储 | 所有数据存储在本地，不联网，不依赖云服务 |
| 6.2 | 存储格式 | JSON 文件（轻量，易备份） |
| 6.3 | 数据目录 | 程序目录下或用户目录下的固定文件夹 |

---

## 七、不需要的功能（明确排除）

- AI 聊天助手
- 云同步 / ICS 日历导入
- 黑名单网站功能
- 任何需要网络连接的功能

---

## 八、UI 风格参考

- 浅色背景（白色/浅灰色系）
- 主界面布局：左侧为日历 + 番茄钟，右侧为备忘录四宫格
- 数字倒计时要大而醒目
- 小悬浮窗：极简，只显示剩余时间 + 当前段类型，半透明浅色背景，可拖动后固定

---

## 九、技术约束

- 轻量化优先，只需支持 Windows
- 用户不需要安装复杂运行时（最好是单文件或简单安装包）
- 推荐技术栈：Python + PyQt6（轻量、可打包为单 exe）
